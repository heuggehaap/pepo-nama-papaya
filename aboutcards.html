Карточки

Это полноценное CRUD-приложение для изучения
иностранного языка с помощью карточек, которое позволит:

  • создавать, удалять, изменять карточки, а также упорядочивать их
    с помощью перетаскивания
  • импортировать карточки в приложение из файла и экспортировать их в файл
  • отслеживать прогресс (функция самооценки)
